# md2json

MarkdownファイルをネストされたJSON構造に変換するDenoスクリプトです。見出しの階層をネストされたオブジェクトとして保持し、リスト項目からキーと値のペア（メタデータ）を抽出します。

## 機能

- Markdownの見出し（`#`、`##` など）をネストされたJSONのキーに変換します。
- 本文のテキストを `text` プロパティに集約します。
- コロンを含む箇条書きのリスト項目（例: `- key: value`）をキーと値のペアとして解析します。
- 依存関係のない、シンプルで単一ファイルのDenoスクリプトです。

## 必要条件

- [Deno](https://deno.land) ランタイム

## 使い方

Markdownファイルを変換するには、Denoでスクリプトを実行します。例えば、提供されている `words.md` ファイルを変換するには以下のコマンドを実行します。

```bash
deno run --allow-read --allow-write https://code4fukui.github.io/md2json.js words.md
```

このコマンドは `words.md` を読み込み、同じディレクトリに構造化された `words.md.json` ファイルを生成します。

## 変換ルール

このスクリプトは、いくつかのシンプルなルールに従ってMarkdownの要素をJSON構造にマッピングします。

### 1. 見出しとテキスト

見出し（`#`、`##`、`###` など）はJSONオブジェクトのキーになります。見出しのレベルによってネストの深さが決まります。見出しに続くテキストはすべて `text` プロパティに集約されます。

**入力 (`example.md`)**
```markdown
# Title

This is the main body text.

## Subtitle 1

Body text for the first subtitle.

## Subtitle 2

Body text for the second subtitle.
```

**出力 (`example.md.json`)**
```json
{
  "Title": {
    "text": "This is the main body text.",
    "Subtitle 1": {
      "text": "Body text for the first subtitle."
    },
    "Subtitle 2": {
      "text": "Body text for the second subtitle."
    }
  }
}
```

### 2. メタデータとネストされた見出し

コロンを含む箇条書きのリスト項目（`- key: value`）は、キーと値のペアとして解析されます。これはメタデータを追加する際に便利です。見出しのレベルが深くなると、より深くネストされたオブジェクトが作成されます。

**入力 (`words.md` より)**
```markdown
## インクルーシブ教育システム

人間の多様性の尊重等を強化し、障害者が精神的及び身体的な能力等を可能な最大限度まで発達させ、自由な社会に効果的に参加することを可能とするとの目的の下、障害のある者と障害のない者が共に学ぶ仕組み

- 出典: [次期学習指導要領等に向けたこれまでの審議のまとめ](courseofstudy2020.md)

### 要件

- 障害のある者が一般的な教育制度から排除されないこと
- 自己の生活する地域において初等中等教育の機会が与えられること
- 個人に必要な「合理的配慮」が提供されること等
```

**出力 (`words.md.json` より)**
```json
{
  "インクルーシブ教育システム": {
    "text": "人間の多様性の尊重等を強化し、障害者が精神的及び身体的な能力等を可能な最大限度まで発達させ、自由な社会に効果的に参加することを可能とするとの目的の下、障害のある者と障害のない者が共に学ぶ仕組み",
    "出典": "[次期学習指導要領等に向けたこれまでの審議のまとめ](courseofstudy2020.md)",
    "要件": {
      "text": "- 障害のある者が一般的な教育制度から排除されないこと\n- 自己の生活する地域において初等中等教育の機会が与えられること\n- 個人に必要な「合理的配慮」が提供されること等"
    }
  }
}
```
